
# QR SaaS – Free QR Code Generator

![MIT License](https://img.shields.io/badge/license-MIT-green)
![Free Forever](https://img.shields.io/badge/free-forever-brightgreen)

A modern, privacy-friendly, and 100% free QR code generator. No signup, no ads, no tracking. Built with Next.js, React, and Tailwind CSS.

## Features
- Generate QR codes for URLs, text, and contact info (vCard)
- Multi-language support (English, Spanish)
- Download QR as SVG
- Copy QR data to clipboard
- Confetti animation on download
- Responsive, accessible, and mobile-friendly
- No data stored, no ads, no tracking
- Open source (MIT License)

## SEO & Meta
- Optimized for search engines (robots.txt, sitemap.xml, meta tags)
- Social sharing ready (Open Graph, Twitter Card)

## Pages
- Home: QR generator
- About: Project info, mission, credits
- Contact: Contact form, email

## Getting Started

1. **Install dependencies:**
	```bash
	npm install
	```
2. **Run locally:**
	```bash
	npm run dev
	```
3. **Deploy:**
	- Push to GitHub
	- Import to [Vercel](https://vercel.com/)
